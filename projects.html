University Thesis: Game Prototype

The projects presented in this video is my thesis project which I developed independently in my last year of university. 2D Game Engine

The next project was a 2D space shooter which I also created during my last year of university. I considered this project particularly educational as not only it laid the foundation of my C++ programming, but it also taught me important programming principles such as polymorphism, encapsulation, inheritance and dependencies. This project was about creating a game engine using a shell provided by our lecturer.

Besides the programming techniques described above, I also improved my knowledge about game development architectures, how important is the architecture of a game in both development and further expansion, as well as how to approach a project of this size.

Automation Tool

This section presents the interface for the automation tool that I created to improve my day-to-day efficient and reduce the number of repetitive tasks. This was one of my first projects in Python and I feel like it helped me to understand the capabilities of this programming language.

The first tool is designed to take a specific block of text, split it into different lines, parse the lines based on key words, then generate a note with only the key information alongside automatically-generated data based on the input.

The second one is a password generator which is designed to create a password based on specific date-based parameters.
